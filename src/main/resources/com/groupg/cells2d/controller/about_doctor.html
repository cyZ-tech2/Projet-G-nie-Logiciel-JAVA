Cells2D — Interface Médecin

v1.0  PGL ING1-GI 2025–2026

Description

L'interface médecin permet d'observer en temps réel l'évolution d'une épidémie sur Paris et d'enregistrer des cas patients. La simulation utilise un modèle SEIRS pré-configuré avec les paramètres du COVID-19 (variant original).

Paramètres COVID-19 appliqués : β = 0.25 • σ = 0.2 • γ = 0.1 • μ = 0.02 • ξ = 0.02 • 1 step ≈ 1–2 jours

Contrôles

  • ▶ — Lancer la simulation
  • ⏸ — Mettre en pause (sauvegarde automatique)

Enregistrer un cas patient

  1. Cliquer sur la cellule concernée sur la carte (surbrillance bleue)
  2. Sélectionner le groupe d'âge
  3. Saisir les symptômes (séparés par des virgules)
  4. Saisir le nombre de cas infectés dans cette cellule
  5. Cliquer sur Ajouter le cas

L'ajout d'un cas transfère N individus du compartiment S vers I dans la cellule sélectionnée. L'état visuel de la cellule est mis à jour immédiatement. La simulation est sauvegardée automatiquement dans ~/.cells2d/doctor_autosave.dat.

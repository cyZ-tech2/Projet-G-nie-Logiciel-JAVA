Cells2D — Interface Chercheur

v1.0  PGL ING1-GI 2025–2026

Description

L'interface chercheur permet de simuler la propagation d'une épidémie sur une grille 2D représentant Paris, à l'aide d'un modèle SEIRS (Susceptible → Exposed → Infected → Recovered → Susceptible).

Contrôles

  • ▶ — Lancer la simulation
  • ⏸ — Mettre en pause
  • ⏹ — Réinitialiser
  • ⏭ — Exécuter un pas
  • ⏮ — Revenir au pas précédent (Rewind)

Modes de dessin

  • Cursor — Sélectionner une cellule pour afficher ses statistiques
  • Brush — Changer l'état d'une cellule au clic
  • Bucket — Changer l'état de tout un arrondissement

Paramètres SEIR

  • β — Taux de transmission
  • σ — Taux d'incubation (1/durée d'incubation)
  • γ — Taux de guérison (1/durée de maladie)
  • ξ — Déclin de l'immunité / nouveaux variants
  • μ — Taux de mortalité
  • p — Taux de propagation géographique

Fichiers

Cells2D Simulation

Version: 1.0

Project: PGL ING1-GI 2025-2026

Description

This application simulates epidemic propagation on a 2D grid using an SEIR model.

User Guide

  • Start: start the simulation
  • Pause: pause the simulation
  • Step: execute one simulation step
  • Save: save the simulation
  • Open: load a saved simulation
  • New: create a new simulation
  • Close: close the application
